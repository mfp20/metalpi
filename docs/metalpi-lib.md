# metalpi-lib

metalpi-lib is a C++ helper library, part of the [Metal-Pi platform](https://github.com/mfp20/metalpi).

## Hardware
* ARM
* DPI
* GPIO
* HVS
* OTP
* PV
* PLL
* POWER
* DDR2
* SDHOST
* SMI
* Timer
* VEC
* VPU


## System
* Memory
* Cores
* Threads

* [GPIOd](docs/gpiod.md),
* GPGPUd
